# Linktree Ops

This organisation contains automation workflows for the **IT** and **People & Culture** teams at Linktree.

## Repos

| Repo | Team | Purpose |
|---|---|---|
| [it-automations](https://github.com/linktree-ops/it-automations) | IT | Okta, Kandji, Kisi, Lumos workflows |
| [people-automations](https://github.com/linktree-ops/people-automations) | People & Culture | BambooHR, Employment Hero workflows |
| [ops-shared](https://github.com/linktree-ops/ops-shared) | All | Shared reusable workflows and scripts |

## Running a workflow

1. Go to the relevant repo
2. Click **Actions** in the top nav
3. Select the workflow from the left sidebar
4. Click **Run workflow** → fill in any inputs → **Run workflow**

> All credentials are stored as GitHub Secrets — you never need to handle API keys directly.

## Getting help

Ping `#it-team` or `#people-culture` in Slack, or open an issue in the relevant repo.
